This endpoint retrives all the family members of a patient.

GET HTTP Request

GET https://ept.praavahealth.com/API/PatientPortal/MyFamilyList?token=<token>&upi=<upi>

Request Information

Category Value
HTTP Request GET
URL https://ept.praavahealth.com/API/PatientPortal/MyFamilyList?token=&upi=

Headers

No Headers required

Input Schema of request

Parameter Description Required
token Mandatory token for authorization. true
upi UPI of the individual patient. true

Output Schema of request

Filed Description
success status of the request.
result json array of all the family members of a patient.
message message of the response .
